# DEAD.INC
## Web.py
### Requirement

1. Web.py (http://webpy.org/install)
2. Pygal (http://www.pygal.org)
3. iso3166 (https://pypi.python.org/pypi/iso3166/0.6)

### Run server
1. Run command
```
# Default port is 8080 and online on http://localhost/
python deadinc.py <port>
```
